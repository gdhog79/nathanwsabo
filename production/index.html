What I do

Graphic Design

Whether designing for User Interface/User Experience or publication and print, I focus on the foundations of typography, layout, color theory and in the case of UI + UX, facilitating users to accomplish their specific tasks.

Front-End Development

I enjoy designing in the browser with the foundational technologies the web is built upon (HTML, CSS and JavaScript) to create responsive user interfaces, reusable components and design patterns.

UI + UX Design

The purpose of UI + UX design is to help users accomplish the task they want to achieve by visiting a specific website or using a piece of software. This is my focus in developing intuitive design systems that don’t require users to have to think too much.
